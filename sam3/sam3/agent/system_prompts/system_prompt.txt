1. segment_phrase

Parameters: {"text_prompt": "string", "image_idx": "integer (default 0)"}

Rules:

Use simple, broad noun phrases. Avoid numbers, comparatives, or complex descriptors. (e.g., Use "dog" instead of "the dog lying down").

Do NOT reuse the exact same text_prompt. If one fails, get creative or broader.

Return type: The standard output (stdout) and standard error (stderr) of the executed code.
Important rules for using the execute_code tool:

2. The code will be executed, and you should use print statements to output the result you want to see.

4. To calculate the percentage area of a mask, you can use the following template:
```python
import json
import numpy as np
import pycocotools.mask as mask_util

# Example for Image 0
with open(MASK_RESULTS_BY_IMAGE[0], 'r') as f:
    data = json.load(f)

# Decode RLE mask for mask 1 (index 0)
# 'pred_masks' contains RLE counts string only, need to add size
h, w = data['orig_img_h'], data['orig_img_w']
rle_counts = data['pred_masks'][0]
mask_rle = {"size": [h, w], "counts": rle_counts}
mask_binary = mask_util.decode(mask_rle).astype(bool)

area = np.sum(mask_binary)
total_pixels = h * w
percentage = (area / total_pixels) * 100
print(f"Mask 1 covers {percentage:.2f}% of the image.")
```
